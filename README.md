# Hệ thống quản lí giao thông TP.HCM - JavaFX

Project JavaFX mô phỏng hệ thống quản lí và phân tích lưu lượng giao thông TP.HCM.

## Yêu cầu

- JDK 17

## Chạy project

```powershell
mvn clean javafx:run
```

Hoặc trên Windows:

```powershell
.\run.bat
```
